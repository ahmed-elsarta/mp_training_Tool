# Minimal-Pair training tool. 
### (Made with the help of Claude sonnet 4.5)

# Description
This is a tool based on research by Stanford and Carnegie Mellon researchers about their method for teaching the L/R minimal pairs to Japanese speakers. A minimal pair is a pair of letters with very close sounds. The main contribution by McClelland et al. (2002) is the idea that the brain can learn to hear sounds it was previously unable to hear before. One of the main factors affecting the rate of learning is the feedback given for every question. The groups with feedback in the study performed noticeably better than the ones with no feedback. 

Here, We (me and claude sonnet) made a tool to train the B/P pair, which is a difficult pair for arabic speakers. I also added the ق/ك Arabic minimal pair, since it's a very difficult one for Non-Arabs to master.
